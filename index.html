About

Agentex is a Chrome extension that uses Google's Gemini AI to intelligently tailor your resume to specific job descriptions. It helps you pass ATS (Applicant Tracking System) screenings and land more interviews by optimizing your resume content.

The extension works exclusively with LaTeX resume files, ensuring professional typesetting while leveraging AI to replace projects, enhance skills, and refine experience descriptions based on the target job requirements.

Get Started

1

Install the Extension

Add Agentex to Chrome from the Chrome Web Store or load it from source for development.

2

Configure API Key

Get a free Gemini API key from Google AI Studio and enter it in the extension settings.

3

Upload Resume & Job Description

Upload your LaTeX resume file and paste the target job description to begin optimization.

4

Generate & Download

Click generate to let AI tailor your resume, then download the optimized version as PDF.

For development installation:

git clone https://github.com/sbeeredd04/Agentex.git cd Agentex # Load tailored-resume-extension in chrome://extensions/
